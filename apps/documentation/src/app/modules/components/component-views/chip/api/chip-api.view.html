API

Import
import { BlizzChipComponent } from '@blizz/ui';
BlizzChipComponent
Selector: bzz-chip
Inputs
Property Type Default Description
variation string - Sets the visual variation of the chip.
selectable boolean true Determines whether the chip can be selected or not.
selected boolean false Gets or sets the selected state of the chip. Only applicable when the chip is selectable.
disabled boolean false Gets or sets the disabled state of the chip.
Outputs
Property Event Type Description
selectionSelect void An event emitted when the chip is selected.
selectionDeselect void An event emitted when the chip is deselected.
selectionChange boolean An event emitted when the selection state of the chip changes. It emits a boolean value indicating the new selection state.
disabledChange boolean An event emitted when the disabled state of the chip changes. It emits a boolean value indicating the new disabled state.
