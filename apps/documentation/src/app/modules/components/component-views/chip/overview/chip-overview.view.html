Overview

The BlizzChipComponent is designed to be a versatile component for displaying information or triggering actions. It offers a visually appealing chip interface that can be customized with different variations. The chip can be selectable, allowing users to toggle its state, and it can also be disabled to prevent interaction.

Usage
<bzz-chip>Text</bzz-chip>
Selectable

The chip component can be made selectable by setting the selectable input to true. When the chip is selectable, users can toggle its state by clicking on it. The selected state can be controlled programmatically using the selected input. 

<bzz-chip [selectable]="true">
  Selectable Chip
</bzz-chip>
