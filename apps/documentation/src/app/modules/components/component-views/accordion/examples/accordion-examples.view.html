Multi Accordion

You can set the multi attribute to allow multiple expansion panels to be open at the same time.

<bzz-accordion [multi]="true">
  <bzz-expansion-panel>
    <bzz-expansion-panel-trigger>Panel 1</bzz-expansion-panel-trigger>
    <bzz-expansion-panel-content>Content 1</bzz-expansion-panel-content>
  </bzz-expansion-panel>

  <bzz-expansion-panel>
    <bzz-expansion-panel-trigger>Panel 2</bzz-expansion-panel-trigger>
    <bzz-expansion-panel-content>Content 2</bzz-expansion-panel-content>
  </bzz-expansion-panel>
</bzz-accordion>
