Overview

The "Accordion" component is a wrapper for Expansion Panel components that provides accordion behavior.
By default, only one expansion panel can be expanded at a time. However, you can enable multi-panel expansion by setting the multi input to true.

Usage

To use the Accordion component, simply wrap your expansion-panel components inside the bzz-accordion element. You can customize the behavior of the accordion by using the available component inputs. 

<bzz-accordion>

  <bzz-expansion-panel>
    <!-- Content of the first expansion panel -->
  </bzz-expansion-panel>

  <bzz-expansion-panel>
    <!-- Content of the second expansion panel -->
  </bzz-expansion-panel>

</bzz-accordion>
