API

Import
import < BlizzAccordionComponent > from '@blizz-ui/ui';
BlizzAccordionComponent
Selector: bzz-accordion
Inputs
Input Type Default Description
group string - The group identifier for the accordion. If not provided, it defaults to the component's ID.
multi boolean false Specifies whether the accordion allows multiple panels to be expanded at the same time.
id string - The ID attribute of the element.
variation string | null null The variation of the accordion.
Computed Properties
Property Type Description
panelComponentsRefs QueryList<BlizzExpansionPanelComponent> A query list of references to the expansion-panel components.
Methods
Method Return Type Description
expandAll() void Expands all expansion panels in the accordion.
collapseAll() void Collapse all expansion panels in the accordion.
setupPanels() void Sets up the nested expansion panels properties, providing accordion behaviour.
