API

Import
import { BlizzExpansionPanelComponent } from '@blizz/ui';
BlizzExpansionPanelComponent
Selector: bzz-expansion-panel
Inputs
Property Type Default Description
group string - Sets the group identifier for the expansion panel. Panels with the same group identifier will behave as an accordion, where only one panel can be expanded at a time.
expandSingle boolean false Determines whether only one panel can be expanded at a time within the group. This property is only applicable when a group identifier is set.
expanded boolean false Gets or sets the expanded state of the panel. When set to true, the panel will be expanded and its content will be visible. When set to false, the panel will be collapsed and its content will be hidden.
id string - The ID attribute of the element.
variation string | null null Sets the visual variation of the expansion panel.
Computed Properties
Property Type Description
collapsed boolean Returns whether the expansion panel is currently collapsed. This property is computed based on the expanded state of the panel.
Methods
Method Return Type Description
toggle() void Toggles the expanded state of the panel. If the panel is expanded, it will be collapsed, and vice versa.
expand() void Expands the panel, making its content visible.
collapse() void Collapse the panel, hiding its content.
BlizzExpansionPanelTriggerComponent

The component that represents a header element of the expansion panel. It is the element that users interact with to expand or collapse the panel. This component typically contains text, buttons, or icons that provide a visual cue and indicate the state of the panel. By default, it contains a chevron arrow representing whether panel is expanded or collapsed.

Selector: bzz-expansion-panel-trigger
Host Directive: BlizzCdkExpandableTriggerDirective
Inputs
Name Type Description
canExpand boolean Specifies if the trigger can trigger expansion.
canCollapse boolean Specifies if the trigger can trigger collapse.
Outputs
Name Event Type Description
indicatorClick MouseEvent Event emitted when the indicator is clicked.
Computed Properties
Name Type Description
directive BlizzCdkExpandableTriggerDirective The instance of the host directive BlizzCdkExpandableTriggerDirective.
service ExpansionPanelService The instance of the ExpansionPanelService.
config BlizzConfigExpansionPanel The configuration object for the associated panel.
indicatorStyles - The styles for the indicator component from config.
defaultIndicatorStyles - The styles for the default indicator component from config.
hovering boolean Indicates whether the trigger is currently being hovered over.
BlizzExpansionPanelContentComponent

The component that represents a content of the expansion panel. It is the section that is shown or hidden when the panel is expanded or collapsed, respectively. This component can contain any type of content, such as text, images, or other components, to provide additional information or functionality related to the panel.

Selector: bzz-expansion-panel-content
Host Directive: BlizzCdkExpandableDirective
Computed Properties
Property Type Description
directive BlizzCdkExpandableTriggerDirective The instance of the host directive BlizzCdkExpandableTriggerDirective.
service ExpansionPanelService The instance of the ExpansionPanelService.
ExpansionPanelService

A service that provides methods and properties to control and manage the expansion panel functionality.

Properties
Name Type Description
triggerRef BlizzExpansionPanelTriggerComponent The reference to the trigger component.
contentRef BlizzExpansionPanelContentComponent The reference to the content component.
hostRef BlizzExpansionPanelComponent The reference to the host expansion panel component.
expanded boolean Indicates whether the panel is expanded or not.
group string | symbol | number | null The group identifier for the panel.
expandSingle boolean Indicates whether only a single panel can be expanded at a time.
collapsed boolean Indicates whether the panel is collapsed or not.
Methods
Name Return Type Description
toggle void Toggles the expansion state of the panel.
expand void Expands the panel.
collapse void Collapse the panel.
