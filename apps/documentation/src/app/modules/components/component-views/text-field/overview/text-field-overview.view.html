Overview

The text field component is a wrapper for native input-related elements such as input itself, label, and custom elements like prefix and suffix. It provides a convenient way to structure and style these elements together to create a cohesive text field UI.

Usage
<bzz-text-field>
  <label>Label Text</label>
  <input [type]="text" />
</bzz-text-field>

The example above demonstrates a text field with a label and an input element. You can add additional elements like prefix and suffix as needed.
