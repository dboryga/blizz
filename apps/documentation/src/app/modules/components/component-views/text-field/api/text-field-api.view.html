API

Import
import { BlizzTextFieldComponent } from '@blizz-ui/components';
BlizzTextFieldComponent
Selector: bzz-text-field
Inputs
Name Type Default Description
id string - The ID attribute of the text field.
variation string | null null The variation style of the text field.
labelPosition Props.InputLabelPosition 'top-left' The position of the label in the text field.
labelFloatingAlways boolean false Indicates whether the label should always float above the input.
Outputs
Name Event Type Description
inputFocus void Emitted when the text field is focused.
inputBlur void Emitted when the text field loses focus.
inputFocusChange boolean Emitted when the focus state of the text field changes.
Computed Properties
Name Type Description
empty boolean Indicates whether the text field is empty.
required boolean Indicates whether the text field is required.
disabled boolean Indicates whether the text field is disabled.
focused boolean Indicates whether the text field is currently focused.
Methods
Name Return Type Description
focus() void Sets focus on the input element.
blur() void Removes focus from the input element.
